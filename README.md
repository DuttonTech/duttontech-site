# duttontech-site

Marketing landing page for DuttonTech (Next.js 15 App Router + Tailwind + Framer Motion).
The DuttonCapital dashboard remains a separate Streamlit app at app.duttontech.com.

## Run locally
1. npm install
2. npm run dev   (opens http://localhost:3000)

## Build
- npm run build

Notes: relative imports only (no @/ alias). Animated components are marked "use client".
